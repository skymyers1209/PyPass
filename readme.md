

# 🔐 PyPass

**PyPass** is a free and open-source password manager built with simplicity, security, and accessibility in mind. It allows users to securely store, manage, and access their passwords in one place.

---

## ✨ Features

* 🔒 Secure password storage
* 🧠 Simple and easy-to-use interface
* ⚡ Lightweight and fast (built in Python)
* 🗂️ Local password management
* 🔑 Designed for everyday users and students

---

## 🚧 Project Status

> ⚠️ **Active Development**

* 🖥️ **Windows:** ✅ Fully supported (Stable)
* 🍎 **macOS:** ⚠️ Not tested (May work, not recommended)
* 🐧 **Linux:** ⚠️ Not tested (May work, not recommended)

While PyPass is written in Python and *may* run on macOS and Linux, these platforms have **not been tested** and are **not officially supported at this time**.

## 🔐 Security Notice

PyPass is designed with security in mind, but:

* Always use a strong master password
* Do not share your database files
* Keep your system secure and up to date

> ⚠️ This project is still evolving. Use responsibly.
